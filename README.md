# houseprice-predition
Comsen实验室任务-波士顿房价预测
项目概述：
        项目来源：kaggle房价预测。
        项目目标：根据已有数据预测房屋销售价格。
技术实现：
        本次项目运用了简单的线性回归模型进行房价的预测
可视化过程：
        本次项目中添加了更多可以直观看出模型训练过程以及误差随训练次数减少的图像。
项目结果：
        所有的可视化结果保存至master分支下。最终误差 log rmse约为0.16
项目收获：
        本次项目让我学会了如何运用线性回归模型进行简单的预测，为以后的Lenet，resnet模型的学习铺垫。
